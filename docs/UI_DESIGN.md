# NoteFlow UI 设计方案

## 项目概述

NoteFlow 是一个简洁高效的笔记应用，采用深色主题设计，提供流畅的笔记编辑体验。

## 技术栈

- React 18
- TypeScript
- Vite
- CSS3

## 整体布局

应用采用经典的双栏布局设计：

```
┌─────────────────────────────────────────┐
│  左侧边栏 (300px)  │   右侧编辑区 (flex)  │
│                    │                      │
│  - 应用标题        │   - 标题输入框       │
│  - 新建按钮        │   - 内容编辑区       │
│  - 搜索框          │                      │
│  - 笔记列表        │                      │
│                    │                      │
└─────────────────────────────────────────┘
```

## 设计风格

### 色彩方案

**主色调**
- 深色背景：`#1a1a1a` (主背景)
- 侧边栏背景：`#252525`
- 卡片背景：`#2a2a2a`
- 悬停背景：`#303030`
- 选中背景：`#1a3a5a`

**强调色**
- 主要强调色：`#4a9eff` (蓝色，用于按钮和选中状态)
- 悬停强调色：`#3a8eef`

**文字颜色**
- 主文字：`#e0e0e0` (浅灰)
- 标题文字：`#ffffff` (白色)
- 次要文字：`#999999` (中灰)
- 辅助文字：`#666666` (深灰)
- 占位符：`#555555`

**边框颜色**
- 分隔线：`#333333`
- 输入框边框：`#444444`
- 焦点边框：`#4a9eff`

### 字体系统

- 字体家族：`-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', sans-serif`
- 标题字号：32px (编辑区标题)、24px (应用标题)、16px (笔记标题)
- 正文字号：16px (编辑器)、14px (按钮/搜索框)、13px (预览)、12px (日期)
- 行高：1.6 (编辑器内容)

### 圆角设计

- 按钮/输入框：6px
- 笔记卡片：8px
- 滚动条：4px

## 组件设计

### 1. 左侧边栏 (Sidebar)

**尺寸**
- 宽度：300px
- 背景色：`#252525`
- 右边框：1px solid `#333`

**子组件**

#### 1.1 侧边栏头部
- 应用标题：📝 NoteFlow (24px, 白色)
- 新建笔记按钮：
  - 全宽按钮
  - 背景色：`#4a9eff`
  - 悬停：`#3a8eef`
  - 圆角：6px
  - 内边距：10px

#### 1.2 搜索框
- 占位符：🔍 搜索笔记...
- 背景色：`#1a1a1a`
- 边框：1px solid `#444`
- 焦点边框：`#4a9eff`
- 圆角：6px

#### 1.3 笔记列表
- 可滚动区域 (flex: 1)
- 笔记卡片：
  - 背景色：`#2a2a2a`
  - 悬停：`#303030`
  - 选中：`#1a3a5a` + 蓝色边框
  - 圆角：8px
  - 内边距：15px
  - 间距：8px
  - 包含：标题、预览、日期

### 2. 右侧编辑区 (Editor Area)

**布局**
- Flex: 1 (占据剩余空间)
- 背景色：`#1a1a1a`
- 内边距：30px 40px
- 垂直布局 (flex-direction: column)

**子组件**

#### 2.1 标题输入框
- 字号：32px
- 字重：700 (粗体)
- 颜色：白色
- 背景：透明
- 无边框
- 底部间距：20px

#### 2.2 内容编辑区
- Flex: 1 (占据剩余空间)
- 字号：16px
- 行高：1.6
- 颜色：`#e0e0e0`
- 背景：透明
- 无边框
- 无 resize

#### 2.3 空状态
- 居中显示
- 提示文字：选择一条笔记开始编辑
- 颜色：`#666`
- 字号：18px

## 交互设计

### 笔记列表交互
- **点击**：选中笔记，切换到编辑模式
- **悬停**：背景色变化 (`#303030`)
- **选中状态**：蓝色背景 + 蓝色边框
- **滚动**：自定义滚动条样式

### 搜索功能
- **实时搜索**：输入时即时过滤
- **搜索范围**：标题 + 内容
- **大小写不敏感**

### 编辑功能
- **实时保存**：输入时自动更新
- **标题编辑**：点击标题输入框直接编辑
- **内容编辑**：点击内容区域直接编辑
- **自动更新时间**：编辑后更新 updatedAt

### 新建笔记
- 点击"+ 新建笔记"按钮
- 创建新笔记并自动选中
- 默认标题："新建笔记"
- 默认内容：空

## 动画效果

- 按钮悬停：0.2s 过渡
- 笔记卡片：0.2s 过渡 (背景色、边框)
- 输入框焦点：即时变化

## 滚动条样式

- 宽度：8px
- 轨道背景：`#252525`
- 滑块背景：`#444`
- 滑块悬停：`#555`
- 圆角：4px

## 数据结构

```typescript
interface Note {
  id: number          // 唯一标识
  title: string       // 笔记标题
  content: string     // 笔记内容
  updatedAt: string   // 更新时间 (YYYY-MM-DD)
}
```

4. ✅ 实时搜索
